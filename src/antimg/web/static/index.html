      Акции = усредняющая пирамида: стоп от средней, риск всего стека ровно b на любом шаге.

Прикрытый Интрадей (Korovin, методика /hedgedintraday) — это
      НЕ антимартингейл. Длинный синтетический стреддл = 2 ATM колла − 1 фьючерс (длинная гамма,
      макс. убыток = уплаченная премия), тету которого оплачивает контр-трендовый скальпинг
      по экспоненциальной сетке. Стреддл MtM по Блэку-Шоулзу, роллится в ATM у экспирации.
      Скальп-модель «grid» (по умолчанию) — событийная сетка на ДНЕВНОМ такте: шаг ≈ дневной ATR,
      ордера исполняются по пути бара (OHLC), круговые сделки книжатся, реально залипшие части
      маркируются и закрываются на ролле. Длинные опционы (6–12 мес) = медленная тета ⇒ дневной
      бар РЕПРЕЗЕНТАТИВЕН, а не нижняя оценка. («range» — старая интрадей-эвристика-нижняя-граница.)
      Судим по совокупному росту счёта (стреддл = гамма−тета · скальп). Цель доктрины ≈ 25–40%/год.

Смотри, как ПИ исполняется на выбранном окне (напр. 3 месяца). Цена + полоса
      Боллинджера (детектор «флета»), страйк ATM-стреддла (ступенька), КАЖДЫЙ контр-трендовый вход скальпа
      (🔻шорт у верха / 🔺лонг у низа) и выход (○), роллы (◆), и разложение P&L.
      Правила видно глазами: на пробое полосы новые контр-трендовые входы НЕ ставятся (не фейдим тренд —
      стреддл и «третий-резерв» бегут); залипшие части переносятся; «тренд как ад» → стреддл-гамма даёт плюс,
      несмотря на застрявшую ⅓. На сильном тренде ИТОГО положительный, хоть скальп и в минусе — это by design.

Чистый стреддл — БЕЗ интрадей-скальпа, БЕЗ раннего ролла. Каждый период тратим
      выбранный % депозита на ATM-стреддл (колл+пут по текущей цене), держим до экспирации,
      получаем внутреннюю стоимость |S_T−K|, и роллим в новый стреддл. Видно, во сколько обходится стреддл
      («аренда» = премия) и зарабатывает ли он, если просто держать до конца.
      
⚠ Данные: отдельного API не нужно. Цена базового актива — реальная (дневная история);
      премия опциона — модельная цена Блэка-Шоулза по моделируемой IV (реализованная воля / VIX-поверхность),
      а НЕ котировка с реального опционного фида. Выплата на экспирации |S_T−K| — по реальной цене. Поскольку
      IV обычно ≥ реализованной (премия за риск волатильности), покупка-и-удержание стреддла обычно −EV — это и проверяем.

Колл vs Пут по отдельности. Тот же сетап, что и чистый стреддл, но КАЖДАЯ нога —
      ОТДЕЛЬНАЯ стратегия: каждый период покупаем либо только ATM-колл, либо только ATM-пут (на % депозита),
      держим до экспирации, роллим. Видно по каждой ноге: сколько раз в плюсе/в минусе и серии прибылей/убытков
      подряд. Колл выигрывает на росте выше своей премии, пут — на падении ⇒ их серии почти зеркальны.
      
⚠ Премия — модель Блэка-Шоулза по IV (как в Tab 10), не котировка реального фида. Каждая нога считается
      на своём отдельном банке (как если бы вы торговали ТОЛЬКО коллы / ТОЛЬКО путы).
